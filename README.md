# Comercial Ferreira - Showroom Digital Otimizado

Este projeto foi desenvolvido para resolver o gargalo logístico e de vendas da **Comercial Ferreira**, substituindo catálogos em PDF pesados por uma plataforma web ultra-rápida.

## Demo ao Vivo
Acesse a versão hospedada (produção):

https://allandevindie.github.io/ferreira-showroom/index.html

**Compatibilidade:** testes recomendados em Chrome, Edge ou Safari em dispositivos móveis e desktop.

---

## 🚀 Diferenciais Estratégicos
- **Performance Mobile:** Focado em carregar instantaneamente em conexões móveis.
- **Catálogo Inteligente:** Estrutura preparada para integração com Google Sheets (Planilhas), permitindo atualização em tempo real sem mexer no código.
- **Filtros Dinâmicos:** Facilita a busca do cliente por categorias (Ana Ruga, Suede, Suplex).
- **Conversão Direta:** Cada produto leva o cliente ao WhatsApp do vendedor com o contexto do item desejado.

## 🛠️ Manutenção via Google Sheets
O grande valor deste projeto é a manutenção via planilha:
1.  O administrador atualiza os dados (Preço, Estoque, Fotos) em uma planilha do Google.
2.  O site reflete as mudanças automaticamente via API.
3.  **Resultado:** Independência total para o cliente e zero suporte técnico para o desenvolvedor.

## 📁 Estrutura
- `index.html`: Home focada em autoridade e dor do cliente.
- `catalogo.html`: Vitrine de produtos otimizada.
- `vendedores.html`: Hub de atendimento humanizado.
- `css/style.css`: Identidade visual premium.
